Muuv

Live|Code

Muuv is a personal workout journal geared towards cyclists and runners. Users can create update and save statistics and routes related to their daily workouts.

It takes full advantage of the Google Maps API to provide users with the ability to create routes, and obtain specific information about the routes, with simple clicks on a map.

Bubble Burst

Live|Code

Bubble Burst is a bubble shooter game. The objective is to cluster bubbles of the same color, resulting in them bursting and being removed from the board. This verision of the game is played 'survival style', meaning the bubbles keep coming. The game is over when either the player has managed to clear the board or the bubbles have reached the bottom.

For me Bubble Burst was a first dive into collision logic and a refresher in trigonometry.

January 2018
JavaScript, Canvas

Competitive Ant Racing

Live|Code

A very special app for purveyors of the competitive ant racing scene, who wish to view statistics related to competing ants before races take place. Uses a GraphQL API to source ant racer data. Odds are calculated randomly and at different intervals. React was used to display calculation updates in real time.
